## Cold open — 0:00–0:25

*[P1 opens Home page fresh, boot sequence plays while talking]*

**P1:** Good morning, judges. While that plays — this is our site booting up. We didn't want to just build a website *about* a network that woke up, we wanted it to feel like you're logging into one.

## The concept — 0:25–1:45

**P1:** Every page shares this status bar — uptime, node count, signal, all ticking live no matter where you go. You're never on four separate pages, you're inside one interface, just moving around it.

**P2:** It's built entirely from scratch — plain HTML, CSS, JavaScript, no framework. Full control over how every interaction felt.

---

## Manifesto — 1:45–2:45

*[P1 clicks Manifesto]*

**P1:** This is the philosophy page. The theme is about the line between human thought and machine logic disappearing — so it's literally built into our color system. Magenta is human, cyan is machine, and violet — this color — is where the two stop disagreeing. Same three colors on every page, including the network graph.

## Network — live demo — 4:30–6:00
